Detailed Description

Interface for AzFramework::EntityContextRequestBus, which is the EBus that makes requests to a given entity context. If you want to make requests to a specific entity context, such as the game entity context, use the interface specific to that context. If you want to make requests to multiple types of entity contexts, use this interface.
