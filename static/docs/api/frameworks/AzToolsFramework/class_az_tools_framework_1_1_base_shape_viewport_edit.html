Detailed Description

Base class for viewport editing of shapes. This class and its descendants are intended to be viewport independent, so avoid using buses addressed by EntityId or EntityComponentIdPair. To facilitate uses in the main viewport, an EntityComponentIdPair can optionally be specified using AddEntityComponentIdPair, in order to hook manipulators up to undo/redo behavior and UI refreshing.

Member Function Documentation

void AzToolsFramework::BaseShapeViewportEdit::AddEntityComponentIdPair ( const AZ::EntityComponentIdPair &  entityComponentIdPair)

Optionally used to associate an EntityComponentIdPair with the shape manipulators. This is useful in the main editor viewport for hooking up undo/redo behavior and UI refreshing. This should be called after Setup. Otherwise, the manipulators will not have been created yet.

void AzToolsFramework::BaseShapeViewportEdit::InstallBeginEditing ( AZStd::function< void()>  beginEditing)

Install optional functions to provide hooks for connecting to undo/redo systems, refreshing UI etc. These are not expected to be necessary in the main editor viewport, because the manipulators handle those things automatically, but they may be useful in other viewports.
