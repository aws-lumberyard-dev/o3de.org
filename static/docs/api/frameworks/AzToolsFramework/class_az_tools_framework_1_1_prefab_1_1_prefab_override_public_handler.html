Checks whether overrides are present on the given entity id. Overrides can come from any ancestor prefab but this function specifically checks for overrides from the focused prefab.

Parameters
entityIdThe id of the entity to check for overrides.
relativePathFromEntityThe relative path from the entity. This can be used to query about overrides on components and their properties
Returns
true if overrides are present on the given entity id from the focused prefab.
