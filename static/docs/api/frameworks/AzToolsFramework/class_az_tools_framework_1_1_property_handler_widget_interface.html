Detailed Description

Base class for all property editor widgets in the DocumentPropertyEditor. Property handler widgets are registered to the PropertyEditorToolsSystemInterface and instantited as part of the DocumentPropertyEditor, with one handler instance being constructed for each property editor.

static constexpr bool AzToolsFramework::PropertyHandlerWidgetInterface::IsDefaultHandler ( )
inlinestatic

If overrideen, this can be used to indicate that this handler should be added to the "default" pool of property handlers. Default property handlers will have ShouldHandleNode queried for PropertyEditor nodes that don't have an explicit Type set. For example, a numeric spin box might be registered as a default handler, in which case a node like: <PropertyEditor value="5" valuetype="int"> would resolve to a spin box, while <PropertyEditor type="Slider" value="5" valuetype="int"> would still resolve to a slider.
