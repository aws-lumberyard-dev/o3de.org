static void AzQtComponents::AzQtApplication::InitializeDpiScaling ( )
static

Initializes Qt DPI scaling to handle displays with high display densities, such as Retina displays. Currently, this uses Qt's system DPI awareness, in which a common device scaling factor will be calculated across all attached screens.

Warning
This must be called before this AzQtApplication instance is initialized.
