template<class Visitor , class... Variants>
static constexpr auto AZStd::variant_detail::visitor::impl::make_dispatch_for_all ( )
inlinestaticprivate

This function makes (m^n) visitor functions where m is the number of variants and n is the number of alternatives per variant This tree of visitors can invoke the visitor for any combination of alternatives. For three variants with three alternatives 27 functions are generated For two variants with three alternatives 9 functions are generated For three variants with two alternatives each 8 functions are generated
