Extracts information on the provided pointer and moves it to the content of the pointer. If FullyProcessed is returned there's no more information to process and if ContinueProcessing is returned the elementClassData will be updated to the element the pointer was pointing to and storeTypeId will indicate if the type needs to be explicitly written.
